Mark, may I?

Want to explore the Facebook API without all the setup hassle? Just choose your permissions below, click "Create New Token", and you'll be asked to authorize this app. Then you can use the generated token to make API calls.

Now you can paste the token into Facebook's Graph API Explorer or try out a manual API call.


Extended Permissions: the user can deny these, so don't count on them being available.


Open Graph Permissions and Page Permissions: like basic permissions, the user must accept these to use your app.
